このパッケージにはReference Toolの内蔵ハードディスクからゲームアプリ
ケーションを起動する為に必要な以下のユーティリティ及びドキュメントが
含まれています。

  - HDD起動ゲームユーティリティ
  - NPDRMパッケージ作成ツール

内蔵ハードディスクからゲームアプリケーションを起動するには、NPDRM
パッケージ作成ツールを用いて適切なNPDRMパッケージを作成してください。
作成したNPDRMパッケージはシステムソフトウェア上の
「ゲーム -> Install Package File」メニューからインストールする事が
出来ます。

HDD起動ゲームアプリケーションの開発手順については、「HDD起動ゲームユー
ティリティ概要」を、ゲームアプリケーションの暗号化の方法や、内蔵ハード
ディスクへのインストール方法の詳細は、「NPDRMパッケージ作成規定」を参照
ください。

PLAYSTATION(R)3 Programmer Tool Runtime Library SDK 154.001環境に
本パッケージを展開してご使用ください。
