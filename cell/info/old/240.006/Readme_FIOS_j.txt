このパッケージは、FIOS（ファイルI/Oスケジューラー）ライブラリに関する
ファイルが含まれています。

FIOS は、ファイル I/O 要求のスケジューリング機能、データ圧縮機能、
データキャッシュ機能を提供し、ゲームのファイルアクセスを高速化する
ライブラリです。
詳細につきましては、パッケージ内のドキュメントを参照してください。

SDK240.001の関連パッケージを展開した環境で、本パッケージを展開して
ご使用ください。

必要環境
----------------------------------------------------------------------
- 本リリースにおいて、FIOS のファイル圧縮機能を使用する際には、
  PLAYSTATION(R) Edge release 0.4.1 を以下の URL からダウンロードし
  インストールしてください。

  https://ps3.scedev.net/projects/psedge

----------------------------------------------------------------------
制限事項
----------------------------------------------------------------------
- PSARC-Users_Guide_j(e).pdf 内に圧縮アルゴリズムとして LZMA に関する
  記述がありますが、現在の psarc プログラムでは Zlib のみがサポート
  されています。

- libfios リファレンスには、処理が完了したとき op::getRequestedSize() 
  と op::getFulfilledSize() は同じ値を返すとありますが、異なる値を返す
  場合があります。

- 作成済みのディレクトリに対し、scheduler::readDirectory() または
  scheduler::readDirectorySync() を行った後、そのディレクトリを削除し、
  再度同じ名前のディレクトリを作成すると、そのディレクトリに対する 
  scheduler::readDirectory()、scheduler::readDirectorySync() が正しく
  動作しない場合があります。

- op::waitMultipleTimeout() は、指定した時間にタイムアウトしない問題が
  あります。op::waitMultipleDeadline() をご利用ください。
