変更点
----------------------------------------------------------------------
<新規機能>
- シェーダの出力においてfloat型以外の型をサポートするオプション
  -nofloatbindings を追加しました。

- テクスチャユニットに指定するテクスチャ形式のデフォルト設定を可能に
  しました。-texformatオプションの後に指定する<texture unit number>に
  "default"を指定して下さい。
    例)sce-cgc -texformat default <format string>

<不具合修正>
- 以下の不具合を修正しました。

  - sce-cgcが最適化を行う際に、テクスチャのフェッチ命令を必要以上に
    生成する事があります。この事により若干、パフォーマンスが低下する事が
    あります。

- 以下の不具合を修正しました。

  - -contallocオプションを指定してシェーダをビルドした際にsampler*型の
    配列がTEXUNIT*へ連続して割り当てられない事があります。

- 以下の不具合を修正しました。

  - フラグメントシェーダ内にてh1texcompare2D()を使用した際に正しい結果が
    得られない事があります。

- 以下の不具合を修正しました。

  - シェーダへの入力パラメータの属性を誤って"unreferenced"に設定して
    しまう事がありました。
    なお、"unreferenced"が設定されたパラメータはsce-cgcstripによって
    削除されてしまいます。その為、sce-cgcstripを通したシェーダが正しく
    動作しない事があります。

- 以下の不具合を修正しました。

  - cgbファイルを作成した際にバーテックスプログラム中にて一度も使用されて
    いない、初期値付きの定数が残ってしまう不具合があります。

- 以下の不具合を修正しました。

  - sce-cgcのプロファイルオプションであるOutColorPrecが無効になっていた為、
    オプションに指定しても無視されてしまいます。

- 以下の不具合を修正しました。

  - シェーダをコンパイルする際の最適化のロジックに問題がありfloat値の
    符号が反転してしまう事があります。
    この不具合の影響により期待した描画が行われない事があります。

- 以下の不具合を修正しました。

  - シェーダー内にて複合代入演算子("*=", "+="等)を用いた場合に、正しい
    計算結果が得られない事があります。
    例)DiffuseColor.xyz *= DiffuseColor.www;
