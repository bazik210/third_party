このパッケージは、SDK280.001 パッケージ (PS3_SDK-280_001.zip) の
make_package_npdrm コマンド(cell\host-win32\bin\make_package_npdrm.exe)
に不具合修正を行ったものです。

SDK280.001の関連パッケージを展開した環境で、本パッケージを展開して
ご使用ください。

----------------------------------------------------------------------

+---host-win32
     +---bin
          +---make_package_npdrm.exe
