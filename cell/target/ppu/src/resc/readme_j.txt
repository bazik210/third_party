含まれています。

----------------------------------------------------------------------
libresc ソースのディレクトリ構成
----------------------------------------------------------------------
resc/
  |---readme_*.txt
  |---libresc.sln                  (Visual Studio 2005 用 solution)
  |---src                          (ソースディレクトリ)
  |    |---libresc.vcproj          (Visual Studio 2005 用 project)
  |    +---Makefile
  |---include                      (libresc.h)
  +---lib                          (Makefileでビルドしたライブラリが入る)

----------------------------------------------------------------------
ビルド方法
----------------------------------------------------------------------
- Windows (VisualStudio 2005)
　libresc.sln を開いてビルドしてください。
　"PS3 Release"/"PS3 Debug" の２つの構成があります。

- Linux / Windows (msys)
　src ディレクトリにて、
  $ make
  してください。以下の様にすると、デバッグビルドが得られます。
  $ make CELL_RESC_DEBUG=1

- ランタイムライブラリとしての動作保証につきましては以下が条件となります。
